First open a Terminal Session; this is where we'll be using the commands. Use command "CD" to get into the correct location/directory. So "CD documents". Command "LS" lists the folders within the "documents" folder; Ours is "newGitTest" Folder within "documents". So we would CD documents>CD newGitTest Go to your gitbhub.com page and copy the link to your repository. Ex: https://github.com/mjkoza5/newGitTest.git

Git Terminal Commands:

  1. git clone https://exmaple/path.git
  2. git status: Will show you new files and untracked files. Ex: index.html
  3. git add < file > to add any new, uncommited files. EX: git add index.html. Adds the file to our project
  4. git commit: means you're locking in this file to your repository. add "-m" to add message. Example: "git commit -m page2.html"
  5. git push: this syncs and pushes the file index.html to your repository on github.com
    6. So now if you go to github.com, refresh the page, you'll see the file "index.html"
  6. git pull: will pull files from github.com

**If you accidently type "git commit", you enter a screen that seems stuck and you cannot get out. To get out, hit "ESC" then type ":wq".
