The first thing you want to do is create a new file called "main.css" or "whateveryouwant.css". This file is where you will keep your CSS code for your website. This is called EXTERNAL CSS. Within this file you will apply STYLE to your webpages. You take your HTML tags, which are called "selectors" (such as < body >) and you apply a "rule" to them within brackets (such as Background:red;)"


Here is an Example:

To pair the CSS with your webpage, in the Header Tag, insert < link href="whatever.css" rel="stylesheet" />.


Example:

Sometimes you may need to specify where your "whatever.css" file is located within the link tag. Example: lets say my "whatever.css" file is in the "home" Folder. SImply input "home/whatever.css". SO it would look like this:
< link href="home/whatever.css" rel="stylesheet" />
